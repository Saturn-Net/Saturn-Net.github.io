About Saturn

Saturn is a decentralized network allowing developers to builds Dapps quickly and effortlessly unlike similar projects which are slow and expensive to maintain. The main motive for developing Saturn was accessibility not just for the end users but for developers.

To make a decentralized Identity system using distributed ledger technology was something Saturn wanted to incorporate into the network. The problem was we wanted to use Uport.ID which is great in theory but impractical. Fees due to smart contracts needing Gas was a problem not to mention the security regarding smart contracts on ethereum. This is why Saturn would like to announce Saturn.id

Saturn works with the Stellar Blockchain to deliver smart contracts to users with fees less than a penny.
